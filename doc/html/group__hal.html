Standard HAL Functions

The standard HAL implementation here does not pass the device handle to the HAL functions; it is stripped out by the HAL macros. If that capability is needed, define ADI_HAL_USE_DEVICE_HANDLE and implement the required HAL function prototypes in the port-specific headers.

HAL Wrappers

These macros will strip out the device handle before passing arguments to the HAL functions. If the HAL implementation for a certain platform requires use of the device handle to implement HAL functions, define ADI_HAL_USE_DEVICE_HANDLE and supply definitions for the HAL function prototypes in the port-specific headers.

◆ HAL_FcsCalculate()

uint32_t HAL_FcsCalculate ( uint8_t *  pBuf,
uint32_t  nbBytes 
)

HAL Frame check sequence (FCS) calculation.

Parameters
[in]pBufPointer to data buffer on which to perfrom the calculation.
[in]nbBytesNumber of bytes on which to perfrom the calculation.
Returns
4-byte Ethernet FCS as U32.

Calculate and return the 32-bit FCS. Per the 802.3 standard, the computation is via CRC32 BZIP2 with polynomial = 0x4C11DB7, initial CRC = 0xFFFFFFFF, and with the CRC post-complemented.
